Name InstructionGroupType
Used by (from the same schema document) Complex Type DirectionsType
Abstract no
Documentation A group of instructions that make up a major top-level section of a pattern. An instruction group may optionally be labelled. Instruction groups may consist of one or more sections or any combination of block operations (including instructions, instruction-refs, and rows). Instruction groups cannot be nested and may only appear directly under the directions element.
