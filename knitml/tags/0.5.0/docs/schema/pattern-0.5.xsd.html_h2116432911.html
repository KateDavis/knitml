Name RowType
Used by (from the same schema document) Complex Type DirectionsType , Complex Type InstructionGroupType , Complex Type InstructionType , Complex Type SectionType
Abstract no
Documentation A row (or round) of knitting. Processing through a row's child elements works all stitches on the current needles sequentially. The end of the row is reached when the last stitch on the needles for this row has been worked. The last operation in this row must put the knitting at the end of the row, unless the 'short' attribute has been set to true.
