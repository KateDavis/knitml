This sample is a swatch of various techniques not covered in other samples.
It shows you how to:
- knit with multiple colors (both stranded / intarsia, and mosaic colorwork)
- define rows which are out of sequence (e.g. rows 1,3,5; then rows 2,4)
- bind off all stitches
- cross stitches (i.e. cabling)
- write a pattern which is not internationalized
